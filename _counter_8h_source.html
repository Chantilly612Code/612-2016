21  * Class for counting the number of ticks on a digital input channel.
22  * This is a general purpose class for counting repetitive events. It can return
23  * the number
24  * of counts, the period of the most recent cycle, and detect when the signal
25  * being counted
26  * has stopped by supplying a maximum cycle time.
27  *

30  */
